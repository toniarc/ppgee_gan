Esta atividade tem como objetivo comparar empiricamente o desempenho de duas arquiteturas de GANs: Vanilla GAN e Wasserstein GAN. Vocês receberão três códigos Python que devem ser executados na sequência indicada abaixo. Todos os códigos contêm documentação extensa e comentários didáticos para auxiliá-los.
 Primeiro, execute o arquivo . Este código criará um dataset artificial de 1000 imagens 16x16 representando três formas geométricas (círculos, quadrados e triângulos). O código gera o arquivo  (que será usado no treinamento) e uma visualização  para você verificar os dados gerados. Tempo estimado: 1 minuto.
 Em seguida, execute o arquivo . Este código implementa e treina tanto a Vanilla GAN quanto a Wasserstein GAN usando o dataset gerado anteriormente. O treinamento roda 100 épocas para cada GAN e salva automaticamente amostras de imagens geradas ao longo do processo nas pastas  e . As métricas de treinamento (perdas, acurácias, tempo) são salvas na pasta . Observe no terminal o progresso de cada época. Tempo estimado: 30-60 minutos dependendo do seu hardware (mais rápido com GPU).
 Após o treinamento completo, execute o arquivo . Este código carrega as métricas salvas e gera automaticamente gráficos comparativos e um relatório textual detalhado na pasta . Você encontrará os arquivos  (com 4 gráficos comparativos),  (análise temporal) e  (relatório com conclusões). Leia atentamente todos os arquivos gerados. Tempo estimado: 10 segundos.
 Após completar a execução baseline (etapas 1-3), você DEVE realizar pelo menos 3 experimentos adicionais modificando hiperparâmetros no arquivo . Os hiperparâmetros estão claramente marcados no início do código (LATENT_DIM, BATCH_SIZE, LEARNING_RATE_G, LEARNING_RATE_D, NUM_EPOCHS, N_CRITIC, WEIGHT_CLIP). Para cada experimento: modifique um ou mais hiperparâmetros, execute novamente o código 2, renomeie as pastas de saída para não sobrescrever os resultados anteriores, execute o código 3 para gerar nova análise, e documente todas as modificações e resultados.
 Após realizar todos os experimentos, elabore um relatório técnico em PDF contendo: introdução com contextualização teórica, fundamentação matemática das duas GANs, metodologia detalhando dataset e hiperparâmetros, resultados e discussão de TODOS os experimentos com gráficos e tabelas, e conclusões comparando o desempenho das duas arquiteturas. A estrutura completa do relatório está detalhada no arquivo  que acompanha os códigos.
 Python 3.7+, PyTorch, NumPy e Matplotlib. Instale com: pip install torch numpy matplotlib
